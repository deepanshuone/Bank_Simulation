# Bank Automation System (Python + SQL)

## Description
A console-based bank management system using Python and SQL to handle
account creation, deposits, withdrawals, and balance checks.

## Technologies Used
- Python
- SQLite / MySQL
- SQL

## Features
- Create account
- Deposit money
- Withdraw money
- Balance enquiry

## Database Schema
(accounts, transactions)

## How to Run
1. Install Python
2. Run: python main.py
